## Langkah untuk menjalankan Dashboard

1. Clone repo github
```
git clone https://github.com/felixanthony03/Data-Analysis-Dicoding-Dashboard
```

2. Install requirements.txt
```
pip install requirements.txt
```

3. Run Streamlit
```
streamlit run Dashboard.py
```
